# AutoCredit Perú

Sistema web para la simulación de créditos vehiculares utilizando el método francés de amortización, desarrollado como proyecto académico.

## Descripción

AutoCredit Perú permite gestionar clientes y realizar simulaciones de financiamiento vehicular considerando:

- Método Francés de amortización.
- Cuota inicial.
- Cuota balón (Compra Inteligente).
- Periodos de gracia.
- Seguros obligatorios.
- Comisiones.
- Cálculo de TCEA.
- VAN y TIR.
- Cronograma completo de pagos.
- Historial de simulaciones.

## Requisitos

Antes de ejecutar el proyecto instalar:

- Node.js 20 o superior
- npm

---

## Instalación

Clonar el repositorio

```bash
git clone https://github.com/GerardoPalacin03/Trabajo-Final202610---AutoCredit.git
```

Ingresar a la carpeta

```bash
cd Trabajo-Final202610---AutoCredit
```

Instalar dependencias

```bash
npm install
```

---

## Ejecutar el proyecto

Modo desarrollo

```bash
npm run dev
```

Luego abrir en el navegador

```
http://localhost:3000
```

## Funcionalidades
